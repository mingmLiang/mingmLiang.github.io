    Vue-router

    Vue-router的作用

    Vue-router 就是路由，地址栏输什么，到什么页面，而且vue-router可以通过html5的history实现单页面应用，不刷新跳转，你切地址，只是页面上的组件的切换。另外vue-router还可以实现页面间传参等其他功能。
